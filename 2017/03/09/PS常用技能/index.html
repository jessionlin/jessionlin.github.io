PS常用技能

1.配套使用的部件：

放大镜与导航器
标尺、吸管和信息工具：注：获取图片长度角度的信息，便于图片的更正

2.裁剪工具：

需要其透视按钮时只需在裁剪工具上点击右键。透视可将立体的某一面转化为平面
同时，可在点击裁剪工具后在上工具栏中自定义取片宽高级透明度，并且可运用在其他照
片及清除

3.打印颜色模式：

CMYK：C：青色 M：洋红 Y：黄色 K：黑色
工业颜色模式：RGB
